# LoadForm — Dispatcher Load Capture Tool

## Overview
A desktop app for Uzbekistan-based logistics dispatchers working remotely with US trucking companies.

The app listens to broker calls (microphone or system audio), transcribes speech in real-time using Deepgram, extracts structured load details via an LLM (Ollama Cloud, OpenAI-compatible API), and generates clean output formatted for drivers via customizable templates.

---

## Target User
- Uzbekistan-based dispatchers working remotely for US trucking/logistics companies
- They communicate with US brokers via phone (RingCentral, Zoom, Teams)
- Need to capture load details quickly while on a call
- Output goes to drivers via SMS/WhatsApp/email

---

## Core Flow
1. Dispatcher opens app before/during a broker call
2. Presses "Start Capture" — chooses Mic or System Audio
3. Audio streams to Deepgram for real-time transcription
4. Transcript sent to LLM (Ollama Cloud) to extract structured fields
5. Fields displayed with confidence indicators (color-coded + percentage)
6. Dispatcher reviews and edits if needed
7. Presses "Copy for Driver" — output formatted via selected template
8. Paste into WhatsApp/SMS/email to driver

## Audio Capture Modes

### Mic Input (FREE)
- Dispatcher repeats broker info into mic
- No legal concerns
- Zero latency, works out of the box

### System Audio (PAID)
- Captures RingCentral/Zoom/Teams audio directly
- Requires user consent and legal acknowledgment
- "Magic" feature — differentiator
- **MUST show legal disclaimer:** "System audio capture may be subject to call-recording laws. You are responsible for ensuring all parties consent."

---

## Confidence UI
- Green border = high confidence (>80%)
- Yellow border = medium confidence (50–80%)
- Red border = low confidence (<50%)
- Percentage badge on each field
- Fields below 70% auto-flagged with "⚠️ Needs Review" badge
- Dispatcher can manually edit any field before copying

---

## Output Templates
- Default template provided out of the box (clean text block)
- Users can create/edit custom templates with variables:
  - `{pickup_location}`, `{pickup_datetime}`, `{delivery_location}`, `{delivery_datetime}`, `{commodity}`, `{equipment}`, `{rate}`, `{weight}`, `{notes}`
- Template preview shown before copying

### Default Template Example
```
LOAD DETAILS
Pickup: {pickup_location} — {pickup_datetime}
Delivery: {delivery_location} — {delivery_datetime}
Rate: {rate}
Weight: {weight}
Equipment: {equipment}
Notes: {notes}
```
